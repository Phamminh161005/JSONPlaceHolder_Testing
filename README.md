# JSONPlaceHolder Testing with PyRestTest

Bộ kiểm thử API được xây dựng bằng **PyRestTest** để kiểm tra các chức năng của Fake API **JSONPlaceholder**.

## Giới thiệu

Repository này chứa tập hợp **30 test case** được viết dưới dạng YAML nhằm kiểm thử các endpoint REST API của JSONPlaceholder.

Mục tiêu của bộ kiểm thử là:

- Đánh giá khả năng kiểm thử API của PyRestTest.
- Kiểm tra các chức năng CRUD cơ bản.
- Kiểm tra các trường hợp dữ liệu hợp lệ và không hợp lệ.
- Phân tích hành vi thực tế của JSONPlaceholder so với chuẩn REST API.
- Phục vụ nghiên cứu và báo cáo học phần Kiểm thử phần mềm.

## Công nghệ sử dụng

- Python 3.x
- PyRestTest
- YAML
- JSONPlaceholder API
